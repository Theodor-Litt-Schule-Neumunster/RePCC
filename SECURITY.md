
# Security Notes

RePCC implementiert derzeit keine umfassenden Sicherheitsmechanismen für produktive oder öffentlich erreichbare Umgebungen. Insbesondere werden keine transportseitigen Sicherheitsstandards wie HTTPS/TLS sowie keine durchgaengige Verschluesselung sensibler Daten sichergestellt; Teile der Kommunikation und Datenverarbeitung können daher effektiv im Klartext erfolgen.

Der Einsatz ist ausschliesslich in kontrollierten, vertrauenswürdigen Test- oder Entwicklungsumgebungen vorgesehen. Eine Nutzung ausserhalb solcher Umgebungen wird ausdruecklich nicht empfohlen.
