# RePCC
Remote Personal Computer Control (RePCC) ist eine App für Android mit der Fähigkeit einen Computer im selben Netzwerk zu steuern.
Unser Ziel für RePCC ist es, eine Android-App für die Steuerung des persönlichen Computers innerhalb eines Netzwerkes mit Hilfe von Flutter zu gestalten. 
Es soll eine Benutzerfreundliche Applikation für Android-Geräte geben und ein Computer-Client, damit beide Seiten reibungslos kommunizieren können.
